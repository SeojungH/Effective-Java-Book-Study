### 진행 방식

---

1. 스터디 레포를 Fork한다
2. 생성된 원격 저장소에 자신의 github ID로 폴더를 생성 (폴더명에 콜론":" 안됩니다!)
3. 생성된 폴더에 챕터별로 소스코드, README 정리본을 업로드
    - commit msg : [챕터] 상세명
    
    ```json
    git commit -m "[3장] 등록/수정/조회 API 만들기"
    ```
    
4. 수요일까지 원본 저장소로 PR 요청
5. 스터디장은 매주 **수요일 스터디 시간 전까지** 각 PR 확인 후 merge 진행
